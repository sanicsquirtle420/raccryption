# RacCryption
A simple text file encrypter / decrypter.

## How to use:
Dependencies: `gcc cmake `

### Flags
* `-e` or `--encrypt`: is used to encrypt a file
* `-d` or `--decrypt`: is used to decrypt a file

```bash
# Compiling the code
make build

# Running the code
./raccryption [FLAG] [FILE_PATH] [KEY]
```
